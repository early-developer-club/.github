# 🤖 Vibe Coding Club

> **"Vibe로 연결되고, 코드로 구현하는 메이커들의 아지트"** > Vibe Coding Club은 영감을 공유하고 함께 결과물을 만들어가는 지속 가능한 성장을 지향합니다.

---

### 🌊 About Us
우리는 단순히 기술을 연마하는 것을 넘어, 서로의 '바이브'를 맞추며 즐겁게 만드는 법을 배웁니다. 직관적인 아이디어를 논리적인 코드로 풀어내고, 그 과정을 기록하며 함께 성장합니다.

---

### 🛠️ Our Activities
Vibe Coding Club은 다음과 같은 4가지 핵심 활동을 통해 메이커들의 성장을 돕습니다.

| Activity | Description |
| :--- | :--- |
| **🎨 Vibe Pick** | **영감의 큐레이션.** 트렌디한 기술, 디자인 레퍼런스, 유용한 도구들을 선정하여 공유합니다. |
| **📚 Vibe Archive** | **지식의 기록.** 프로젝트 과정에서 얻은 인사이트와 학습 데이터를 체계적으로 아카이빙합니다. |
| **🚀 Vibe Build** | **실전 프로토타이핑.** 아이디어를 빠르게 실체화하고 협업을 통해 완성도 있는 서비스를 만듭니다. |
| **☕ Vibe Networking** | **사람과의 연결.** 커피챗과 온/오프라인 모임을 통해 다양한 메이커들과 교류합니다. |

---

### 🔍 Interests & Maker Toolkits
우리는 AI를 활용한 **극강의 효율화**와 **누구나 만들 수 있는 문화**에 집중합니다.

* **🤖 AI Agent & Dev Efficiency:** 개발자가 AI Agent를 커스텀 세팅하여 실무 프로세스를 자동화하고, 더 빠르고 정확하게 제품을 빌드하는 최적의 워크플로우를 탐구합니다.
* **⚡ Business Automation:** 비개발자도 AI 도구를 활용해 자신의 비즈니스 프로세스를 자동화하고, 운영 리소스를 혁신적으로 줄이는 방법을 공유합니다.
* **🎬 Creative GenAI:** LLM부터 영상, 이미지, 사운드 생성 AI까지, 최신 생성형 AI를 활용하여 상상하는 모든 형태의 콘텐츠를 구현합니다.
* **🚀 Hyper-making:** 복잡한 기술적 장벽을 AI로 극복하고, 아이디어를 즉시 시장에 내놓을 수 있는 '초고속 제작(Rapid Prototyping)' 환경을 지향합니다.

### 🤝 How to Join
Vibe Coding Club의 문은 언제나 열려 있습니다.

1. **Explore:** [Repositories](https://github.com/orgs/Vibe-Coding-Club/repositories)에서 진행 중인 프로젝트를 확인해 보세요.
2. **Contribute:** 오픈소스 기여, 이슈 제보, 코드 리뷰 등 모든 형태의 기여를 환영합니다.
3. **Contact:** 함께하고 싶다면 공식 채널이나 LinkedIn을 통해 연락주세요!
